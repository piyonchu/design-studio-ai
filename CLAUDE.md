# Agent Notes

## Design Context

Use `PRODUCT.md` for product strategy, register, users, anti-references, and design principles.

Use `DESIGN.md` for the visual system: CanonForge is a product/tool UI with the north star "The Canon Control Room", obsidian surfaces, signal teal primary actions, quiet indigo secondary accents, compact Geist typography, functional glass containment, and WCAG AA / reduced-motion expectations.

For UI work, preserve provenance, canon, review state, export readiness, and lineage as first-class product information. Do not make the app feel like a generic AI image playground, SaaS marketing page, or Figma/v0-style layout generator.
